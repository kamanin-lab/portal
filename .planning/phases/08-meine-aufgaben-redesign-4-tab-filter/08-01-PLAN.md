# Phase 8 — Plan 08-01: Meine Aufgaben Redesign — 4-Tab Filter System

## Goal

Replace MeineAufgabenPage flat list with a 4-tab filter system. Task grid uses 2-column layout. Tab chips with count bubbles.
TicketsPage (Aufgaben) is NOT changed — RecommendationsBlock stays there.

---

## Success Criteria

1. MeineAufgabenPage renders 4 filter tab chips: Nachrichten, Kostenfreigabe, Warten auf Freigabe, Empfehlungen
2. Each tab chip shows a count badge reflecting tasks in that category
3. Clicking a tab shows only that category's tasks/recommendations in a 2-column grid below
4. Default active tab is the first tab with count > 0
5. Empty state appears when the active tab has zero items
6. Full empty state (no tasks needing attention) appears when all tabs are zero
7. TicketsPage (Aufgaben) is unchanged — RecommendationsBlock stays there
8. Existing tests pass (or are updated to reflect new structure)
9. MeineAufgabenPage stays < 150 lines; MeineAufgabenFilters extracted as a separate component

### Step 1 — Create `MeineAufgabenFilters` component

File: `src/modules/tickets/components/MeineAufgabenFilters.tsx`

```typescript
// Tab filter chips for MeineAufgabenPage
// 4 tabs: unread, awaiting_approval, needs_attention, recommendations
// Each tab: pill chip with label + count badge
// Active: bg-accent text-white; Inactive: bg-surface border-border
// Props: activeTab, onTabChange, counts (object with count per tab)
```

Types:
```typescript
export type MeineAufgabenTab = 'unread' | 'kostenfreigabe' | 'freigabe' | 'empfehlungen'

interface TabCounts {
  unread: number
  kostenfreigabe: number
  freigabe: number
  empfehlungen: number
}

interface Props {
  active: MeineAufgabenTab
  onChange: (tab: MeineAufgabenTab) => void
  counts: TabCounts
}
```

Tab labels (German):
- `unread` → "Nachrichten"
- `kostenfreigabe` → "Kostenfreigabe"
- `freigabe` → "Warten auf Freigabe"
- `empfehlungen` → "Empfehlungen"

Chip style: `flex items-center gap-1 px-2.5 py-1.5 text-xs font-medium rounded-full border transition-colors cursor-pointer whitespace-nowrap`

Count badge: `min-w-[16px] h-[16px] px-1 rounded-full text-2xs font-bold flex items-center justify-center`
- Active: `bg-white/25 text-white`
- Inactive: `bg-surface-raised text-text-tertiary`

Render as horizontal scrollable row on mobile (`overflow-x-auto`), wrapping on desktop.

### Step 2 — Rewrite `MeineAufgabenPage`

File: `src/shared/pages/MeineAufgabenPage.tsx`

Derive tab counts from tasks + taskUnread + recommendations (snoozed excluded):
```typescript
const counts = useMemo(() => ({
  unread: tasks.filter(t => (taskUnread[t.clickup_id] ?? 0) > 0).length,
  kostenfreigabe: tasks.filter(t => mapStatus(t.status) === 'awaiting_approval').length,
  freigabe: tasks.filter(t => mapStatus(t.status) === 'needs_attention').length,
  empfehlungen: recommendations.length, // already excludes snoozed
}), [tasks, taskUnread, recommendations])
```

Default active tab logic (run once when data loads, not reactive):
```typescript
const [activeTab, setActiveTab] = useState<MeineAufgabenTab | null>(null)

useEffect(() => {
  if (!isLoading && activeTab === null) {
    const defaultTab = TAB_ORDER.find(tab => counts[tab] > 0) ?? 'unread'
    setActiveTab(defaultTab)
  }
}, [isLoading, counts, activeTab])
```

Filtered items per active tab:
```typescript
const visibleTasks = useMemo(() => {
  if (!activeTab) return []
  switch (activeTab) {
    case 'unread': return tasks.filter(t => (taskUnread[t.clickup_id] ?? 0) > 0)
    case 'kostenfreigabe': return tasks.filter(t => mapStatus(t.status) === 'awaiting_approval')
    case 'freigabe': return tasks.filter(t => mapStatus(t.status) === 'needs_attention')
    case 'empfehlungen': return recommendations
    default: return []
  }
}, [activeTab, tasks, taskUnread, recommendations])
```

Render structure:
```
<ContentContainer width="narrow" className="p-6 max-[768px]:p-4">
  {/* Header */}
  <div>Meine Aufgaben + total count badge</div>
  <p>subtitle</p>

  {/* Loading */}
  {isLoading && <LoadingSkeleton />}

  {/* All-zero empty state */}
  {!isLoading && totalCount === 0 && <EmptyState />}

  {/* Tab filters */}
  {!isLoading && totalCount > 0 && (
    <MeineAufgabenFilters active={activeTab} onChange={setActiveTab} counts={counts} />
  )}

  {/* Task grid — tabs: unread, kostenfreigabe, freigabe */}
  {!isLoading && activeTab !== 'empfehlungen' && (
    <div className="grid grid-cols-1 md:grid-cols-2 gap-3 mt-4">
      {visibleTasks.map((task, i) => (
        <motion.div key={task.clickup_id} custom={i} variants={cardVariants} initial="hidden" animate="visible">
          <TaskCard task={task} unreadCount={taskUnread[task.clickup_id] ?? 0} onTaskClick={openTask} />
        </motion.div>
      ))}
      {visibleTasks.length === 0 && activeTab !== null && (
        <div className="col-span-2"><EmptyState message="Keine Aufgaben in dieser Kategorie" /></div>
      )}
    </div>
  )}

  {/* Empfehlungen tab */}
  {!isLoading && activeTab === 'empfehlungen' && (
    <div className="grid grid-cols-1 md:grid-cols-2 gap-3 mt-4">
      {recommendations.map((task, i) => (
        <motion.div key={task.clickup_id} custom={i} variants={cardVariants} initial="hidden" animate="visible">
          <RecommendationCard task={task} onTaskClick={openTask} onSnooze={snoozeRecommendation} />
        </motion.div>
      ))}
      {recommendations.length === 0 && (
        <div className="col-span-2"><EmptyState message="Keine Empfehlungen" /></div>
      )}
    </div>
  )}

  {/* Task detail sheet */}
  <TaskDetailSheet taskId={activeTaskId} onClose={closeTask} tasks={tasks} isTasksLoading={isLoading} />
</ContentContainer>
```

Total count in header badge = sum of all 4 tab counts (deduplicated? No — a task with unread + needs_attention appears in 2 tabs, badge counts unique tasks that appear in ANY tab):
```typescript
const totalCount = useMemo(() => {
  const ids = new Set([
    ...tasks.filter(t => (taskUnread[t.clickup_id] ?? 0) > 0).map(t => t.clickup_id),
    ...tasks.filter(t => { const s = mapStatus(t.status); return s === 'needs_attention' || s === 'awaiting_approval' }).map(t => t.clickup_id),
    ...recommendations.map(t => t.clickup_id),
  ])
  return ids.size
}, [tasks, taskUnread, recommendations])
```
