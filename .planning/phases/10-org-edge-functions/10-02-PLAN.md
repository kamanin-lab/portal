| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-10-04 | Elevation of Privilege | Viewer escalation via POST to create-clickup-task / post-task-comment / update-task-status | mitigate | `getUserOrgRole` guard returns 403 BEFORE any ClickUp API call or body parse |

</threat_model>
