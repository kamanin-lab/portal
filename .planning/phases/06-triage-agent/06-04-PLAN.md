To enable WP site audit for a specific client, set wp_mcp_url in Supabase:

```sql
UPDATE profiles
SET wp_mcp_url = 'https://staging.client-site.com'
WHERE email = 'client@example.com';
```

To disable (revert to audit-less triage):
```sql
UPDATE profiles SET wp_mcp_url = NULL WHERE email = 'client@example.com';
```
