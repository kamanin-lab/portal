Vite JavaScript Guide

Key points:
- `vite@7.3.1` requires Node.js `^20.19.0 || >=22.12.0`
- use `loadEnv()` for config-time decisions
- use `import.meta.env` in app code
- keep client-visible env values under `VITE_` prefix
- use `vite preview` only for local preview, not as production hosting
- set `base` explicitly if deploying under a non-root path
