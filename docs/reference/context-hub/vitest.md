Vitest for JavaScript

Key points:
- `vitest 4.0.18` supports Node `^20 || ^22 || >=24`
- default environment is `node`
- use `jsdom` or `happy-dom` for DOM tests
- `globals` defaults to false unless enabled explicitly
- `vi.mock()` is hoisted
- built-in coverage defaults to `v8`
